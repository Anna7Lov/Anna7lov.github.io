1.	name - имя пользователя (строка)
2.	age - возраст поьлзователя (число)
3.	role - роль пользователя (строка)
4.	occupation - род деятельности пользователя (строка)
1.	Операцию, которая вернёт true, если имя пользователя определено И его возраст больше 18
name !== '' && name !== null && name !== undefined && age > 18

2.	Операцию, которая вернёт false, если роль пользователя НЕ "Admin" ИЛИ род деятельности НЕ "CEO".
role === 'Admin' && occupation === 'CEO'

3.	Операцию, которая вернёт род деятельности пользователя, если он определён. Если нет - роль.
occupation ?? role

4.	Операцию, которая вернёт true, если имя пользователя опеределено И ему больше 18, ИЛИ когда его роль "Admin".
name !== '' && name !== null && name !== undefined && age > 18 || role === 'Admin'

5.	Операцию, которая вернёт возраст пользователя, если его роль НЕ "Admin" И имя опредено. В противном случае - false.
name !== '' && name !== null && name !== undefined && role !== 'Admin' && age
